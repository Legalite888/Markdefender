### 1. Host on Vercel (Fastest)
1. **Create a GitHub Repo**: Upload all project files to a new repository.
2. **Link to Vercel**: Sign in to [Vercel](https://vercel.com) and click "Add New Project." Import your GitHub repo.
3. **Environment Variable**: This is the most important step.

   - Value: [Your Gemini API Key from Google AI Studio]
4. **Deploy**: Hit "Deploy" and your professional IP assistant is live!

### 2. Host on Netlify
- Drag and drop the root folder onto [Netlify Drop](https://app.netlify.com/drop).
- Add the `API_KEY` environment variable in the Site Settings.
